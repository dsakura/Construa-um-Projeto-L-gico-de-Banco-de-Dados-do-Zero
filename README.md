# RPG Mestre Manager - Sistema de Gestão de RPG Profissional

## Descrição
Sistema de banco de dados para gestão de campanhas pagas de RPG.

## Estrutura de Arquivos
- `sql/schema.sql`: Script DDL de criação.
- `sql/data.sql`: Script DML para população de dados.
- `sql/3_queries_respostas.sql`: Script DQL com os relatórios analíticos.

## Relatórios Incluídos
1. **Inventário de Personagens:** Cruzamento entre jogadores, personagens e itens.
2. **Campanhas Consolidadas:** Filtro de campanhas com alta frequência de sessões.
3. **Faturamento por Sessão:** Cálculo total de ganhos confirmados.
4. **Ranking de Lucratividade:** Análise financeira por campanha (Atributo Derivado).
5. **Controle de Pagamento:** Identificação de pendências financeiras por jogador.
